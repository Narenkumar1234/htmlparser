# Privacy Policy for HTML Viewer

HTML Viewer does not collect, store, transmit, or share any personal or sensitive user data.

All HTML content pasted by users is processed locally within the browser and is never sent to external servers. The extension does not track browsing history, user activity, or personal information.

If storage is used, it is limited to local browser storage for saving user preferences and does not leave the user's device.

This extension does not use analytics, remote code execution, or third-party tracking services.
